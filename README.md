# Motshelo 💰
Manage your Motshelo group loans, contributions and dividends

## Features
- Member management
- Loan tracking
- Monthly contributions
- Penalty tracking
- Dividend calculations

## Tech Stack
- Backend: Python + FastAPI
- Database: Supabase (PostgreSQL)
- Frontend: React (Vite)
- Hosting: Render + Vercel
